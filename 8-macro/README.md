
# Assignment 8: Macro.

## Task

Implement a `btreemap!` macro, which allows to create [`BTreeMap`](https://doc.rust-lang.org/std/collections/struct.BTreeMap.html) in an ergonomic and declarative way (similarly to `vec!`).

Provide two implementations: one via a declarative macro and the other via a procedural macro.

Prove your implementation correctness with tests.
