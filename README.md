Rust-assignments
Overview
This Rust course is focused on general programming in Rust without any specialization in system or web programming. After completing this course, the student should be able to create efficient and robust software.

Prerequisites
Before starting this course, it is expected that the student read the following chapters from The Rust Book:

Understanding Ownership.
Enums and Pattern Matching.
Generic Types, Traits, and Lifetimes.
Assignments
See subdirectories. Every assignment is located in its own directory with a README.md file and all other needed files.

How to submit an assignment?
Create a private repo.
Add @TheBestTvarynka to collaborators.
Do your task in a separate branch.
Open a new Pull Request into the main branch and assign @TheBestTvarynka as reviewer.
Wait until @TheBestTvarynka approves the PR or requests changes.
If the PR is approved, merge it. If not, and changes are requested, then fix the code and tag @TheBestTvarynka again once it's fixed.
Insert the link in the Classroom in the corresponding task.
